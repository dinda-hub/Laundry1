**Laundry Dinda**

📱 Fitur Utama

- 🔐 **Login & Registrasi Pengguna**
- 🏠 **Tampilan Utama** dengan navigasi ke berbagai menu
- 👥 **Manajemen Pelanggan**
- 👨‍💼 **Manajemen Pegawai**
- 🛠️ **Manajemen Layanan**
- ➕ **Layanan Tambahan**
- 🏢 **Manajemen Cabang**
- 💳 **Manajemen Transaksi**

🛠️ Teknologi yang Digunakan

- Bahasa: **Kotlin**
- Database: **Firebase**
- IDE: **Android Studio Meerkat**
